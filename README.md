Task Manager Telegram Bot

A simple Telegram bot for managing personal tasks.

Features

* Add tasks
* View task list
* Complete tasks
* Clear all tasks
* SQLite database storage

Commands

Start

/start

Help

/help

Add a task

/add Learn Python

View tasks

/tasks

Complete a task

/complete 1

Clear all tasks

/clear

Technologies

* Python 3
* SQLite
* pyTelegramBotAPI

Installation

1. Clone the repository

git clone 

2. Install dependencies

pip install -r requirements.txt

3. Create config.py

TOKEN = “YOUR_BOT_TOKEN”

4. Run the bot

python bot.py
